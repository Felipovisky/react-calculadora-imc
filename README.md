# Calculadora de IMC - React

Projeto desenvolvido em **ReactJS** para cálculo de IMC (Índice de Massa Corporal) com classificação de acordo com a tabela oficial de saúde.

## Funcionalidades

- Entrada de **altura** (em metros) e **peso** (em kg)
- Cálculo do **IMC** com precisão de duas casas decimais
- Retorno da **classificação**:
  - Abaixo do peso
  - Peso normal
  - Sobrepeso
  - Obesidade grau I, II ou III
- Botão para **limpar os campos**
- Interface responsiva e simples

## Tecnologias

- ReactJS
- Vite
- CSS moderno (inline e customizado)
